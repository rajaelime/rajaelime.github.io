Hexo & GitHub page 를 활용한 블로그 운영

작성일 2022-07-26 Edited on 2022-07-27

Hexo를 활용하여 별도의 백엔드 서버가 필요하지 않은 정적 컨텐츠 형태의 블로그 구성
정적 컨텐츠 배포용 서비스로는 GitHub Pages 활용

Hexo 설치

GitHub Pages 배포

  1. username.github.io 이름으로 public repo 생성
  2. hexo-deployer-git 설치
    ``` bash
    $ npm install hexo-deployer-git –save
  3. _config.yml 파일 수정
    ``` bash
    deploy:
    type:git
    repo: # https://username.github.io
    branch: [branch]
    message: [message]
    token: [token]
  4. 배포
    ``` bash
    $ hexo clean && hexo deploy
